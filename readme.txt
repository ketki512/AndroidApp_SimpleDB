Android_Application folder contains:

\Android_Application\Snapshots
-Contains the snapshots of the Android App.

\Android_Application\SourceCode
-Contains the Android application source code.

Instructions to run the code:
-You can Import the project in Android studio and run it using your device.
-We have provided the android apk file \Android_Application\app-debug.apk
You can just put the apk file in an android phone and install the app.
